# Aer Lemmings - Distribution

This is a standalone frontend distribution of Aer Lemmings.

## How to Run

Simply open `index.html` in a modern web browser, or serve the files using any static file server:

```bash
# Using Python
python3 -m http.server 8000

# Using Node.js http-server
npx http-server . -p 8000

# Using PHP
php -S localhost:8000
```

Then open http://localhost:8000 in your browser.

## Deployment

You can deploy this folder to any static hosting service:
- GitHub Pages
- Netlify
- Vercel
- AWS S3 + CloudFront
- Any web server

Just upload the contents of this folder to your hosting service.
